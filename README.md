# PDF Reader

**PDF Reader** is a simple Python project for reading, writing, and processing PDF files.  
It is suitable for learning purposes, small applications, and basic PDF automation tasks.

---

## Features

- Read PDF files
- Extract text from PDFs
- Write text to PDFs
- Merge and split PDF files
- Handle basic PDF metadata

---

## Requirements

- Python 3.8+
- PyPDF2 (or compatible PDF library)
